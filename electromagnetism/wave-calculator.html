Wave speed

For electromagnetic radiation, the wave speed is equal to the speed of light. The speed of light in vacuum is 299,792,458 m/s. It is possible to use scientific notation with this calculator, for example 3.3e19 for 3.3 * 10^19. The calculator uses three formulas: Wave speed = frequency * wavelength. Frequency = 1 / period. Photon energy = Planck's constant * frequency. Planck's constant is equal to 6.626 07 015 * 10^-34 J*s.
